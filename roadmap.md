## Architectural Considerations

### Ingestion Pipeline

```
┌─────────────────┐
│   Source APIs   │  Gutenberg, arXiv, CIA.gov, Sacred-texts.com
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│    Extractor    │  PDFs, HTML, ePub → Raw Text
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│    Processor    │  Clean, chunk, detect verse/prose
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  Sequence Store │  IndexedDB / local JSON
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│     R.I.S.E.    │  Chamber delivery
└─────────────────┘
```

### Tech Stack Additions

| Feature | Technology |
|---------|------------|
| PDF parsing | pdf.js or pdf-parse |
| ePub parsing | epub.js |
| Web scraping | fetch + DOMParser |
| Embeddings | Transformers.js (local) or OpenAI API |
| Persistence | IndexedDB via idb-keyval |
| Audio files | Web Audio API AudioBufferSourceNode |
