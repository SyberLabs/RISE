## THE PREMISE

The interface *is* the first session.

Before any content streams,
the user encounters the interface.

This encounter must:

1. **Induce calm** — lower activation, prepare receptivity
2. **Signal seriousness** — this is not entertainment
3. **Communicate care** — every detail considered
4. **Embody the aesthetic** — structured transcendence, luminous rigor
5. **Teach by example** — the interface *is* Neohumanism

## DESIGN PRINCIPLES

---

### PRINCIPLE 1: DARKNESS FIRST

```
╭─────────────────────────────────────────────────────────────────────╮
│                                                                     │
│                                                                     │
│    The default state is darkness.                                  │
│    Light emerges from darkness.                                    │
│    Not the reverse.                                                │
│                                                                     │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    IMPLEMENTATION                                                  │
│                                                                     │
│    • Background: near-black (#0A0A0C)                              │
│    • Elements emerge through luminosity                            │
│    • No white backgrounds anywhere                                 │
│    • Light is always intentional, never default                    │
│                                                                     │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    RATIONALE                                                       │
│                                                                     │
│    The chamber is dark.                                            │
│    The hypnagogic is dark.                                         │
│    The womb is dark.                                               │
│    Creation emerges from darkness.                                 │
│                                                                     │
│    Bright interfaces overstimulate.                                │
│    Dark interfaces prepare.                                        │
│                                                                     │
│                                                                     │
╰─────────────────────────────────────────────────────────────────────╯
```

### PRINCIPLE 2: STILLNESS AS DEFAULT

```
╭─────────────────────────────────────────────────────────────────────╮
│                                                                     │
│                                                                     │
│    Nothing moves unless it must.                                   │
│    Stillness is the ground state.                                  │
│    Motion is always meaningful.                                    │
│                                                                     │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    IMPLEMENTATION                                                  │
│                                                                     │
│    • No decorative animations                                      │
│    • No loading spinners (use opacity fade)                        │
│    • No bouncing, wiggling, attention-grabbing                     │
│    • When something moves, it matters                              │
│                                                                     │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    ALLOWED MOTION                                                  │
│                                                                     │
│    • Transitions between states (fade, not slide)                  │
│    • Content presentation in Chamber                               │
│    • Breathing indicators (very slow, subtle)                      │
│    • Progress indicators (minimal)                                 │
│                                                                     │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    RATIONALE                                                       │
│                                                                     │
│    Constant motion fragments attention.                            │
│    Stillness cultivates presence.                                  │
│    The interface should feel like a meditation space,              │
│    not a casino.                                                   │
│                                                                     │
│                                                                     │
╰─────────────────────────────────────────────────────────────────────╯
```

### PRINCIPLE 4: SPACIOUSNESS

```
╭─────────────────────────────────────────────────────────────────────╮
│                                                                     │
│                                                                     │
│    Space is not empty.                                             │
│    Space is structural.                                            │
│    Give everything room to breathe.                                │
│                                                                     │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    THE SPACING SYSTEM                                              │
│                                                                     │
│    Base unit: 8px                                                  │
│                                                                     │
│    xs:   8px    (micro adjustments)                                │
│    sm:   16px   (related elements)                                 │
│    md:   24px   (standard separation)                              │
│    lg:   32px   (section breaks)                                   │
│    xl:   48px   (major divisions)                                  │
│    2xl:  64px   (primary landmarks)                                │
│    3xl:  96px   (screen-level spacing)                             │
│                                                                     │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    DENSITY PRINCIPLE                                               │
│                                                                     │
│    When in doubt, add space.                                       │
│    Cramped interfaces feel anxious.                                │
│    Spacious interfaces feel calm.                                  │
│                                                                     │
│    Content should never touch edges.                               │
│    Minimum margin: 48px from screen edge.                          │
│                                                                     │
│                                                                     │
╰─────────────────────────────────────────────────────────────────────╯
```

### PRINCIPLE 5: MINIMAL CHROME

```
╭─────────────────────────────────────────────────────────────────────╮
│                                                                     │
│                                                                     │
│    The interface should disappear.                                 │
│    Reduce chrome to the essential.                                 │
│    Every pixel of UI is a pixel not used for content.              │
│                                                                     │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    IMPLEMENTATION                                                  │
│                                                                     │
│    • No visible window decorations where possible                  │
│    • Navigation appears on hover/need, fades otherwise             │
│    • No persistent toolbars during sessions                        │
│    • Settings and options in dedicated screens, not overlays       │
│                                                                     │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    THE HIERARCHY                                                   │
│                                                                     │
│    1. Content (highest priority, most visible)                     │
│    2. Navigation (medium priority, accessible but quiet)           │
│    3. Controls (lowest priority, hidden until needed)              │
│                                                                     │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    RATIONALE                                                       │
│                                                                     │
│    RISE is not a productivity app.                                 │
│    RISE is a consciousness interface.                              │
│    The interface should feel like a portal,                        │
│    not a control panel.                                            │
│                                                                     │
│                                                                     │
╰─────────────────────────────────────────────────────────────────────╯
```

## COLOR IN THE CHAMBER

```
╭─────────────────────────────────────────────────────────────────────╮
│                                                                     │
│                                                                     │
│    CHAMBER-SPECIFIC COLOR                                          │
│                                                                     │
│    ═══════════════════════════════════════════════════════════     │
│                                                                     │
│                                                                     │
│    During sessions, color becomes dynamic:                         │
│                                                                     │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    TEXT WEIGHT → LUMINOSITY                                        │
│                                                                     │
│    weight: 0.0  →  fog (#9B9BA5)                                   │
│    weight: 0.5  →  cloud (#C5C5CD)                                 │
│    weight: 1.0  →  light (#E8E8EC)                                 │
│                                                                     │
│    Higher weight content appears brighter.                         │
│                                                                     │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    STATE → AMBIENT COLOR                                           │
│                                                                     │
│    Induction   →  threshold tint (very subtle)                     │
│    Installation →  ember tint (warmth)                             │
│    Ingestion   →  chamber tint (cool clarity)                      │
│    Synthesis   →  growth tint (emergence)                          │
│    Recursion   →  threshold + ember blend                          │
│                                                                     │
│    Tint is achieved through very subtle vignette,                  │
│    never through changing the core background.                     │
│                                                                     │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    ENTRAINMENT → BRIGHTNESS OSCILLATION                            │
│                                                                     │
│    For entrainment, background luminosity can                      │
│    oscillate very slightly (+/- 2-3% brightness)                   │
│    at the target frequency.                                        │
│                                                                     │
│    ⚠️  Never oscillate faster than 3Hz.                            │
│    ⚠️  Always offer photosensitivity disable option.               │
│                                                                     │
│                                                                     │
╰─────────────────────────────────────────────────────────────────────╯
```

# MOTION

## Animation Guidelines

---

```
╭─────────────────────────────────────────────────────────────────────╮
│                                                                     │
│                                                                     │
│    M O T I O N   P R I N C I P L E S                               │
│                                                                     │
│    ═══════════════════════════════════════════════════════════     │
│                                                                     │
│                                                                     │
│    CORE PRINCIPLE: STILLNESS FIRST                                 │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    Motion is expensive.                                            │
│    Every animation should justify its existence.                   │
│    When in doubt, don't animate.                                   │
│                                                                     │
│                                                                     │
│    ALLOWED ANIMATIONS                                              │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    1. TRANSITIONS (state changes)                                  │
│                                                                     │
│       Page transitions:       Crossfade, 400ms                     │
│       Element appear:         Fade in, 200ms                       │
│       Element disappear:      Fade out, 150ms                      │
│       Hover states:           All properties, 200ms                │
│       Focus states:           Border/color, 150ms                  │
│                                                                     │
│    2. CONTENT PRESENTATION (Chamber only)                          │
│                                                                     │
│       Text appear:            Fade in, 150ms                       │
│       Text hold:              No animation (stillness)             │
│       Text disappear:         Fade out, 150ms                      │
│       Image transitions:      Crossfade, 300ms                     │
│                                                                     │
│    3. FEEDBACK                                                     │
│                                                                     │
│       Button press:           Scale 0.98, 100ms                    │
│       Success:                Brief glow, 300ms                    │
│       Error:                  Subtle shake, 300ms (rare)           │
│                                                                     │
│    4. AMBIENT (optional, can be disabled)                          │
│                                                                     │
│       Sigil pulse:            Opacity 0.8-1.0, 10s cycle           │
│       Loading pulse:          Opacity 0.5-1.0, 2s cycle            │
│       Entrainment:            Brightness ±3%, variable Hz          │
│                                                                     │
│                                                                     │
│    FORBIDDEN ANIMATIONS                                            │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    • Bouncing                                                      │
│    • Sliding (prefer crossfade)                                    │
│    • Spinning/rotating loaders                                     │
│    • Attention-grabbing pulses                                     │
│    • Parallax scrolling                                            │
│    • Decorative particle effects                                   │
│    • Anything faster than 3Hz (safety)                             │
│                                                                     │
│                                                                     │
│    TIMING FUNCTIONS                                                │
│    ─────────────────────────────────────────────────────────────   │
│                                                                     │
│    Default:         ease (cubic-bezier(0.25, 0.1, 0.25, 1))       │
│    Fade in:         ease-out                                       │
│    Fade out:        ease-in                                        │
│    Ambient:         linear (for continuous effects)                │
│                                                                     │
│                                                                     │
╰─────────────────────────────────────────────────────────────────────╯
```
